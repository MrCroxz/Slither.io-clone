## Slither.io (Clone)

This is a simple clone of the game Slither.io.
It's not a perfect recreation, but it was a fun project I made while I was bored...

Feel free to download it and try it out yourself!

# How to Download and Play

    1. Download the ZIP file from this repository.

    2. Extract the ZIP file to a folder on your computer.

    3. Make sure you have Visual Studio installed.

    4. Open the project in Visual Studio.

    5. Open the Main.html file to start the game.
